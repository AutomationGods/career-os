# Privacy Domain

Owns launch-scope user data export and deletion controls.

## Public-launch scope

- Export authenticated user-owned Career OS data.
- Delete authenticated user-owned Career OS data after typed confirmation.
- Treat jobs, packets, profile facts, master resumes, resume versions, document exports, approvals, events, state projections, and snapshots as private user data.

## Out of scope

- Legal advice.
- External compliance automation.
- Enterprise DPA workflows before counsel review.
